# Expenses - Releases

Public download host for the **Expenses** Android app. The app source code lives in
a separate private repository; this repo holds only release APKs and the small
version manifest read by the in-app updater.

- **`version.json`** - metadata for the latest version: version code/name, APK URL,
  and release notes. The app fetches this on launch to decide whether to offer an
  update.
- **APK downloads** - see the [Releases](../../releases) page.

To install manually, download the latest `Expenses-x.y.apk` from Releases and open
it on your phone. Allow installs from this source when Android asks. Once
installed, the app updates itself.
